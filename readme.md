## 规划

1.  搭建开发环境

    加入开发需要的一些依赖包，并且写一些后期方便开发的基础类。

2.  根据设计图开发应用框架

    完成应用界面的 xml 文件设计、编写，将程序的大致页面划分完成，使应用能够完整运行，暂不加入具体内容。

3.  完成每一个模块

    完成每一个界面的具体内容，使得应用能够基本运行，并能够完成基本的各项功能。

4.  优化改善部分细节

    在完成的基础上，优化每一个模块的细节，修复可能出现的 bug 。
